Lab 1

Description: Lab 1 consists of building three simple html pages that deploy CSS (index.html, about.html, labs.html). Each page must have three elements: header, footer, and navigation. I have combined my header and navigation controls into one for easy browsing between pages.

Personally, my level of experience pertaining to CSS, JS, and HTML is intermediate. There is plenty to learn about, so I may be at that stage for awhile. "Complex" styling includes floating elements, a nested class and id selector, which represents the header\navbar up top, along with several margin and padding properties to make this lab entry layout possible.

With all that being said, Lab 1 provides me with a good fondation to build upon when we dive further into client-side scripting. I look forward to things like JSON and JQuery.
